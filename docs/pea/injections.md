# 封闭系统

> 如: 客服呼叫中心, 整个系统会有固定的座机数, 固定的满承载能力. 这种模式下, 引擎会一直维持着虚拟用户数(链接数, 请求数)在指定值. 不会像开放系统那样不断的注入新的虚拟用户.
